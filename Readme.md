# Claude Code Git 操作学習リポジトリ

このリポジトリは、**Claude Code** を使って Git 操作を学習するための練習用リポジトリです。

## 目的

Claude Code（Anthropic の CLI ツール）を通じて、以下のような Git 操作を実際に体験・学習することを目的としています。

- リポジトリの初期化 (`git init`)
- ファイルのステージングとコミット (`git add`, `git commit`)
- ブランチの作成と切り替え (`git branch`, `git checkout`)
- マージとリベース (`git merge`, `git rebase`)
- リモートリポジトリとの連携 (`git remote`, `git push`, `git pull`)
- その他の Git コマンド

## 使い方

1. Claude Code を起動する
2. このリポジトリのディレクトリで Claude Code に Git 操作を指示する
3. Claude Code が実行した操作を確認・学習する

## 注意事項

- このリポジトリは学習・練習目的のみに使用してください
- 実験的な操作を自由に試すことができます
